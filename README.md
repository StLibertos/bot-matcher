Step-by-Step User Interaction with the Bot
1. Set Your Personality Type
Command: /setpersonality
Purpose: Set your MBTI personality type.
Steps:
Run the /setpersonality command.
A dropdown will appear with different MBTI personality types (e.g., INTJ, ENFP, etc.).
Select your personality type.
The bot will provide a description of your chosen type and save it to your profile.

2. Set Your Hobbies
Command: /sethobbies
Purpose: Set your hobbies based on your interests.
Steps:
Run the /sethobbies command.
Choose a category of hobbies (e.g., Creative & Media, Sports, Technology, etc.).
Once you select a category, the bot will present a list of hobbies within that category.
Select your hobbies, and the bot will update your profile.

3. Set Your Communication Styles
Command: /setcommunicationstyle
Purpose: Define your preferred communication styles.
Steps:
Run the /setcommunicationstyle command.
Choose one or more communication styles from a list (e.g., one-on-one talks, group discussions, storytelling).
The bot will save your selections to your profile.

4. View Your Profile
Command: /myprofile
Purpose: View your personal profile that includes personality type, hobbies, and communication styles.
Steps:
Run the /myprofile command.
The bot will display your profile, showing your personality type with a description, hobbies, and communication styles.

5. View Another User’s Profile
Command: /checkprofile @user
Purpose: Check another user’s profile.
Steps:
Run the /checkprofile @user command, replacing @user with the username of the user whose profile you want to view.
The bot will show the user's personality type, hobbies, and communication styles.

Additional Features
Personality Type Descriptions: The bot offers detailed descriptions of each MBTI personality type.
Private Messages: The bot sends sensitive responses (like setting personality type or hobbies) as ephemeral messages, so only you can see them.

How to Use the Bot
Commands: All commands start with /, making it easy to interact with the bot.
User Profiles: The bot stores your profile in memory while it runs, allowing you to update and view your details at any time.
